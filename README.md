# Boundary Discipline (Lean)

A Lean 4 + mathlib formalisation of a small "Boundary Discipline" kernel:

- Regions as sets
- Boundary operators as structured, compositional operators
- No-gap/no-overlap partitions as correctness constraints
- Theorems about preservation under refinement and operator composition

This repo is intentionally minimal but theorem-bearing.
It is designed to be extended toward:

- RCC8-style relations
- Region algebras and gluing
- Typed boundary constraints
- Auditable traces of constraint composition
